Registre Erros, Dúvidas ou sugestões de Melhoria diretamente no sistema. Acompanhe o andamento sem precisar de e-mail ou WhatsApp.

Notificação de resposta

Notificação visual
✅
Aprovação ou reprovação
Ao receber uma resposta, você decide: OK com a Resposta (fecha o chamado) ou Não OK (devolve com comentário para nova análise).
Feedback fechado

Fluxo do chamado:

Novo → Em Análise → Em Desenvolvimento → Resolvido → Fechado

Se a solução não atender: o chamado volta como Reprovado pelo Usuário para nova análise.

👩‍⚕️ Gestão de Terapeutas

Cadastro completo com fluxo de convite automático por e-mail.

1
Cadastro pelo admin
Admin cadastra o terapeuta com nome, e-mail e especialidades. Nenhuma senha é definida nesse momento.
2
Convite automático
O sistema envia automaticamente um e-mail de convite para o terapeuta criar sua própria senha.
3
Primeiro acesso
O terapeuta clica no link, define a senha e já entra no sistema com acesso aos próprios pacientes e agenda.
🔑

Segurança por design

O sistema usa Row Level Security no banco de dados — cada terapeuta só consegue acessar os próprios dados mesmo que tente contornar a interface. Nenhuma regra de negócio depende apenas do frontend.

  • Terapeutas veem apenas seus pacientes, agendamentos e atendimentos
  • Prontuários respeitam o vínculo terapeuta ↔ paciente
  • Administradores têm visão completa do sistema
  • Toda ação fica registrada no log de auditoria
